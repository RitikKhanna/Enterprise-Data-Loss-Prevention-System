# Enterprise Data Loss Prevention (DLP) System

## Overview

The Enterprise Data Loss Prevention (DLP) System is a Python-based security solution developed during my internship at SPC NXT Consulting Pvt. Ltd. The project focused on monitoring enterprise email activity, processing metadata, and supporting secure data monitoring workflows.

## Problem Statement

Organizations require mechanisms to monitor sensitive information shared through enterprise communication channels. This project explored backend workflows to capture and process email metadata for security monitoring and reporting.

## Key Features

- Enterprise email metadata monitoring
- Microsoft Graph API integration
- Automated backend data processing
- Structured MySQL data storage
- Email activity reporting workflows
- Research on endpoint monitoring approaches

## Technologies Used

- Python
- Microsoft Graph API
- MySQL

## My Contributions

- Developed backend modules for processing enterprise email metadata
- Integrated Microsoft Graph API with database workflows
- Designed backend automation for structured data collection
- Evaluated endpoint monitoring approaches for future DLP enhancements
